# Changelog

## 1.0.0 (2026-04-15)

### Added
- Initial release with CIS database compliance profiles
- MSSQL profiles: 2008, 2012, 2014, 2016, 2017, 2018, 2019, 2022
- Oracle profiles: 11g, 12c, 18c, 19c
- Sybase ASE profiles: 15, 16
- PostgreSQL profiles: 15
- Custom sybase_session_local resource for Sybase connectivity
- Profile naming convention: ssc-cis-{platform}{version}-{cis_version}-{revision}
